# Drizzle ORM - README Reference

**Source:** github.com/drizzle-team/drizzle-orm | **Stars:** High | **License:** Apache 2.0

## Key Patterns to Learn From

- **Bold category claim:** "Headless ORM for NodeJS, TypeScript and JavaScript"
- **Concrete technical specs:** "7.4kb minified and gzipped, fully tree-shakeable with zero dependencies"
- **Exhaustive compatibility list** — every database and serverless platform
- **Ecosystem section** — ORM + Kit + Studio suite
- **"Serverless-ready by design"** positioning
- **Sponsor section** — shows community investment

## What Murai Can Copy

- List every compatible technology explicitly (databases, frameworks, runtimes)
- Show the ecosystem — adapters for Midtrans/Xendit/Postgres/MySQL listed prominently
- Concrete size/performance claims build trust
- "Works with X by design" framing
